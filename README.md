# Sobre

- Projeto de dashboard simples em página web usando Python com bibliotecas Streamlit, Pandas, Plotly.

## Objetivo

- Gerar um dashboard usando como base de dados carregados a planilha "base.xlsx" fornecida com informações já tratadas. Analisadas as informaçãos relacionadas aos tipos de assinatura e lucro gerado com assinaturas extras agregadas anualmente.

## Resolução

- Utilizado Python com Streamlit, Pandas e Plotly ao invés de criar dashboard de forma convencional apenas com o Excel.

## Visualização

- Para visualizar em sua máquina é necessária instalação do Python (versão mais recente).
- Crie um ambiente virtual.
- Após isso, instalar as bibliotecas necessárias com o comando:
  `pip install streamlit pandas plotly.express`
- Para rodar o dashboard no navegador:
- `streamlit run index.py`
